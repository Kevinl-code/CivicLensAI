# CivicLensAI

## Problem

Government notices, scholarship circulars, internship announcements and public schemes are difficult to understand.

Users often miss:
- Deadlines
- Eligibility criteria
- Required documents
- Action steps

## Solution

CivicLens AI is a multi-agent document intelligence platform built using Google ADK.

## Architecture

Coordinator Agent
├── Security Agent
├── Document Agent
├── Deadline Agent
├── Eligibility Agent
├── Action Agent
├── Memory Agent
└── Reminder Agent
